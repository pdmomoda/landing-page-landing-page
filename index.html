Frequently Asked Questions

Does this replace VS Code?
Not necessarily! We work alongside it. You can use our standalone app for the 'heavy lifting' of generation, or use our extension to vibe code directly inside your editor.
Is my code safe? Do you train on it?
Your code is yours. We operate with a 'Local-First' architecture. Your keys and code never touch our training servers. We are just the interface.
Can I use local models like Llama 3?
Absolutely. We integrate seamlessly with Ollama and LM Studio so you can code completely offline with zero API costs.
I'm not a 'coder', is this for me?
Yes! That's exactly who "Vibe Coders" are. If you understand logic and systems but don't have syntax memorized, this tool bridges that gap perfectly.
How does the pricing work?
We offer a generous free tier for individuals. The Pro plan includes team collaboration features and advanced context retention history.
Does it handle huge codebases?
Yes. Our RAG (Retrieval-Augmented Generation) engine chunks your code intelligently, so we can handle repos with hundreds of files without confusing the LLM.
